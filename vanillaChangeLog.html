Change Log

A history of updates to VanillaHTML

Version 1.1.1

Small updates and fixes.

  • Removed hover animation from links
  • Redid version numbering to make more sense for future updates
  • Fixed broken links
  • Added link to Github
  • Added more to documentation (still incomplete)

Version 1.1.0

A list of changes made in 1.1.0

I overhauled the design to be much cleaner, and added some more functionality.

  • Added universal header
    • Added multi-page support. You can do this
  • Added universal footer
    • Supports multiple columns (flex-wrapped)
    • Supports a footer banner (using aside)
  • Redesigned sections and header groups
  • Added aside alert variant for different use cases
  • Updated look of anchor tags (links)
  • Switched font from Roboto to Inter for better readability
  • Better contrast ratios for text/backgrounds
  • Updated spacing between sections and paragraphs
  • Removed hgroup lines
  • Fixed broken html
